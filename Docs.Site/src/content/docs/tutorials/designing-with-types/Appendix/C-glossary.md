## 용어 정의

| 용어 | 영문 | 정의 |
|------|------|------|
| 타입 주도 설계 | Type-Driven Design | 도메인 규칙을 타입 시스템에 인코딩하는 설계 접근법 |
| 판별 공용체 | Discriminated Union | 여러 명명된 케이스 중 정확히 하나인 값을 표현하는 타입 |
| 시맨틱 타입 | Semantic Type | 원시 타입을 도메인 의미로 래핑한 타입 |
| 값 객체 | Value Object | 식별자 없이 값으로만 동등성이 판단되는 불변 객체 |
| 상태 기계 | State Machine | 유한한 상태 집합과 상태 간 전이 규칙으로 정의된 모델 |
| 불법 상태 | Illegal State | 비즈니스 규칙상 존재해서는 안 되는 데이터 조합 |
| Boolean Blindness | Boolean Blindness | boolean 값이 어떤 조건을 나타내는지 타입으로 알 수 없는 문제 |
| 완전성 검사 | Exhaustiveness Check | 패턴 매칭에서 모든 가능한 케이스를 처리했는지 검사 |
| 상태 전이 | State Transition | 한 상태에서 다른 상태로의 유효한 변환 |
| 래핑 | Wrapping | 원시 타입을 도메인 타입으로 감싸는 것 |
